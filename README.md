# Computer Networks Fundamentals

This repository contains foundational networking concepts and lab exercises
completed during my undergraduate studies in Computer Networks.

## Topics Covered
- OSI & TCP/IP Models
- Subnetting and IP Addressing
- TCP vs UDP
- Network Topologies
- Protocol Fundamentals

## Tools & Technologies
- Cisco Packet Tracer
- Linux
- Wireshark (basic)

## Purpose
To demonstrate strong theoretical understanding and clear technical documentation
of core networking concepts.
